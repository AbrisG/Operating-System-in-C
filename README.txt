This is the final assignment for COMP310: Operating Systems ourse. 

It is a simulation of an operating system, comprised of:
 * Shell, interpreter, and shell memory
 * Kernel, RAM, and CPU that supports the concurrent execution to up to 3 programs
 * Memory manager in charge of paging and persistance
 * Non-contigous file system that supports read and write operations to different files

To compile, use the makefile along with the 'make' command.

Type 'help' to get a list of all commmands. Use testfile to see an example of the file system.
